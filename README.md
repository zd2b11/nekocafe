# ネコカフェ マロ

## 構成
```
front-page.php          #   ホーム: トップページ
page-news_all.php       #   ニュース: ニュース一覧
single-news_single.php  #            個別のニュース
page-shop.php           #   ショップ: お店の紹介
page-menu.php           #   メニュー: フードメニューの紹介
page-photo.php          #   フォト: フォトギャラリー
page-contact.php        #   コンタクト: お問い合わせ用入力フォーム
page-contact_conf.php   #              入力フォームの内容確認
page-contact_done.php   #              入力フォーム送信完了
page-contact_error.php  #              入力フォーム送信エラー
page-map.php            #   GoogleMap表示ページ(フッターの「googlemapで見る」ボタンから移動)
404.php                 #   404 Not Foundページ
header.php              #   各ページ共通ヘッダー
footer.php              #   各ページ共通フッター
functions.php           #   WordPressに必要なファイル
index.php               #   WordPressに必要なファイル
style.css               #   WordPressに必要なファイル
util.inc.php            #   関数をまとめておく場所
css/
 ├─ common.css
 ├─ sanitize.css
 ├─ top.css
 ├─ news_all.css
 ├─ news_single.css
 ├─ shop.css
 ├─ menu.css
 ├─ photo.css
 ├─ contact.css
 ├─ contact_conf.css
 ├─ done_error_404.css
 ├─ map.css
 └─ magnific-popup.css
js/
 └─ jquery.magnific-popup.min.js
libs/qd/
 ├─ qdmail.php
 └─ qdsmtp.php
```
